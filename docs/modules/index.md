# Omniflow.id Modules Overview

Selamat datang di halaman Overview Semua Modul Omniflow.id. Di bawah ini adalah daftar lengkap modul-modul yang tersedia di platform Omniflow.id.

## Daftar Modul

1. **HRIS** - Human Resources Information System
   - Manajemen sumber daya manusia komprehensif
   - URL: `/modules/hris`

2. **Commerce** - E-Commerce Solutions
   - Platform toko online terintegrasi
   - URL: `/modules/ecommerce`

3. **XRM** - Extended Relationship Management
   - Manajemen hubungan yang lebih dari CRM tradisional
   - URL: `/modules/xrm`

4. **Accounting** - Sistem Akuntansi
   - Software akuntansi lengkap
   - URL: `/modules/accounting`

5. **Analytics** - Business Intelligence
   - Dashboard dan pelaporan bisnis
   - URL: `/modules/analytics`

6. **URLs** - URL Management
   - Shortener dan tracking link
   - URL: `/modules/urls`

7. **Bizzcard** - Digital Business Card
   - Kartu nama digital dengan QR code
   - URL: `/modules/bizzcard`

8. **PoS** - Point of Sale
   - Sistem kasir untuk retail
   - URL: `/modules/pos`

9. **LMS** - Learning Management System
   - Platform pembelajaran online
   - URL: `/modules/lms`

10. **Telemarketing** - Sales & Call Center
    - Manajemen tim sales dan call center
    - URL: `/modules/telemarketing`

11. **Payment Gateway** - Integrasi Pembayaran
    - Penerimaan pembayaran digital
    - URL: `/modules/payment-gateway`

## Integrasi

Omniflow.id juga menyediakan integrasi dengan berbagai sistem populer:

- **ERP**: Odoo, SAP, Microsoft Dynamics
- **E-Commerce**: Shopify, WooCommerce, Magento
- **Accounting**: QuickBooks, Xero
- **Communication**: Slack, Zoom
- **Automation**: Zapier
- **CRM**: Salesforce

## Fitur Utama Omniflow.id

- Cloud-based dan dapat diakses dari mana saja
- Multi-language support (English, Indonesian, Chinese)
- Mobile app (iOS dan Android)
- Two-Factor Authentication
- Role-Based Access Control
- Activity Logging
- Redis Caching untuk performa optimal
- Job Queue untuk background processing
- RESTful API untuk integrasi kustom

## Teknologi Stack

- **Frontend**: React + TypeScript
- **Build Tool**: Vite
- **Styling**: Tailwind CSS
- **Internationalization**: i18next
- **Icons**: Lucide React
- **Routing**: React Router
- **Deployment**: Vercel
